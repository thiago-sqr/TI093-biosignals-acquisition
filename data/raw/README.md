Para usar este projeto:
1. Baixe os dados do experimento no site: [LINK](https://data.mendeley.com/datasets/t8j8v4hnm4/1)
2. Coloque na pasta `data/raw/original/001`, `002`, etc.

Formato esperado:
- .eeg
- .vhdr
- .vmrk
- arquivos CSV auxiliares
